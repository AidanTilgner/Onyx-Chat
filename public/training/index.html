    Interactive Training

    This feature is only meant for development mode. It will only appear if NODE_ENV is set to "development". This is for security reasons and should not be bypassed.

    Here you can test the model with the text input below. Simply type something, press enter, and you'll get a response in the text box. You will then be allowed to train the model with the response you gave. This will help the model learn how to respond to the input you gave.
